Custom Terminal Commands With Bash

01-09-2023

This guide will show you how to create custom terminal commands with bash. This is very useful to have as it allows you to create shortcuts for commands you use often. Here we will look at how to run a python file using a custom command. Let's get into it!

Step 1: Create a bash script

First, we need to create a bash script. This is a file that will contain any commands we want. We will call our script commands.sh and place it in the root directory for easy access. Either open a code editor or use touch/nano/vim. Here we will use vim.

cd //go to root directory vim commands.sh //create commands.sh

Here is what to put in the file:

#!/bin/bash #!/usr/bin/python3 function plant() { python3 [path to file]/[file].py }

The first line tells the computer to treat this as a bash file. The second line tells the computer where python3 is installed and allows us to call python3. The third line creates a custom function called plant. The fourth line runs the python file [filename].py. You may have to change these directories on another operating system/installation.

Step 2: Make the commands work

Now we need to add our script as a source to our boot file to make these commands accessible. To check which shell you have running type:

echo $0 // macOS echo $SHELL // Linux

Now open .zshrc or .bashrc depending on your shell. Here we will use vim.

vim .zshrc //vim .bashrc

Add the following line to the bottom of the file:

source [path to file]/commands.sh

Step 3: Run the commands

We are not done just yet. First restart your terminal to make sure the commands.sh is loaded. Then try out our custom command by typing: "plant". Here's what it looks like for me.

~ via C v11.0.3-clang via ☕ v14.0.1 [Mon Jan 09 202311:39PM] ❯ plant I'm a plant

My python file has a simple print statement that writes "I'm a plant". You can do anything you want with this. You can also add more commands to the commands.sh file.
