Using TaskWarrior for Terminal Commands

01-10-2023

If you spend a lot of time in terminal, maybe you would like to have your to-do list in terminal as well. Or maybe you just want a lightweight, minimal tool that you can use to keep track of your tasks. In this guide we will explore how to do just that using TaskWarrior a neat package you can install with brew.

Step 1: Install Brew

If you don't have brew installed, you can install it by running the following command in terminal. If you are on another OS (Windows/Linux) you can look at this link for installation: https://docs.brew.sh/Homebrew-on-Linux

export HOMEBREW_INSTALL_FROM_API=1 /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

Step 2: Install TaskWarrior

Now that you have brew installed, you can install TaskWarrior by running the following command in terminal.

brew install task

Step 3: Useful commands

Now that you have TaskWarrior installed, you can start using it. Here are some useful commands to get you started.

task add "task name" due:YYYY-MM-D //adds a task with a due date task list //lists all tasks with their due dates task "id" done //marks a task with a certain id (which is shown with task list) as done task help //a comprehensive list of commands
